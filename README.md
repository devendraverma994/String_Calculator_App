# README

* Clone Project Repository from Github url:- 
  - https://github.com/devendraverma994/String_Calculator_App.git

- bundle install

* Ruby version
  - ruby "3.0.0"

* Rails version
  - rails "7.1.3.3"

* Database creation
  - rails db:create

* For successfully Migrate all the pending migrations
  - rails db:migrate

* To run all the test cases
  - bundle exec rspec spec
